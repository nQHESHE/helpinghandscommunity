# Helping Hands Community Website

## Project Overview

Helping Hands Community Care is a non-profit organisation that supports underprivileged individuals and families through food parcels, clothing donations, educational support, and community outreach programmes.

The purpose of this website is to raise awareness, encourage donations, attract volunteers, and provide information about the organisation and its projects.

## CSS Styling

The website uses an external stylesheet (`style.css`) to ensure a consistent appearance across all pages.

Features include:

- Green, blue, and white colour scheme
- Responsive images
- Styled navigation menu
- Styled forms and buttons
- Typography improvements
- Section layouts with spacing and shadows
- Hover effects

---

## Responsive Design

The website is designed to work on:

- Desktop computers
- Tablets
- Mobile phones

Media queries have been used to adjust layouts for smaller screens.
